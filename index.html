Jquery test crossover plugin

Plugin meant to crossover between values. The only hard requirement is jQuery 1.5+. Usage example:

$('#simple').xover('VALUE')

Working example

Example value change. 1350229130607 Currently changing the timestamp.
This behaviour is generated by:

var interval = setInterval(function() {
    $('#test').xover(new Date().getTime());
}, 1000);

Usage

There are two basic ways to use xover plugin:

.xover(textString, [options])
textString value to give
options A set of key/value pairs that configure the crossover.
All settings are optional.
A default options can be set by overriding $.fn.xover.options

.xover([options])
options A set of key/value pairs that configure the crossover.
All settings are optional.
A default options can be set by overriding $.fn.xover.options
If no textString is set then the value needs to be defined with
value key in options

Configurable

Crossover plugin is fully configurable. Configure can be set either page-wide or instance wide. Page-wide configuration means that the default configuration $.fn.xover.options needs to be overriden. List on option keys:

value value to set Only used if value argument is not defined
default: undefined

elClass class name that all generated span elements will receive
default: xover-el

elChangeClass class name that all temporary span elements will receive
default: xover-el-change

elTransition class name that all span elements that are being animated will receive
default: xover-el-transition

speed speed of animation. Speeds are given in milliseconds; higher values indicate slower animations, not faster ones. The strings 'fast' and 'slow' can be supplied to indicate durations of 200 and 600 milliseconds, respectively. If any other string is supplied, or if the duration parameter is omitted, the default duration of 400 milliseconds is used.
default: slow

offset offset for easing animation.
default: 10px
